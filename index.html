How to use

In the field above, paste the encoded: ... line that the Fixium plugin generates in the snapshots/ folder inside your snapshot .fxm file. For example: 

encoded: eyJ0IjoiMjAyNS0wNi0xNSAyMzozNjoxMCIsIm0iOjI2LCJlIjowLCJwIjowLCJ0cHMiOjIwfQ==

(You can copy & paste this line directly)
